# Clinic Management App

A simple desktop application for managing a clinic, built with Java and JavaFX.  
The app allows basic handling of patients, appointments, and database operations through a graphical interface.

## Features

- Patient management
- Appointment handling
- Database connection (MariaDB / PostgreSQL)
- Password hashing with BCrypt
- Desktop GUI built with JavaFX

## Tech Stack

- Java 21
- JavaFX
- Maven
- MariaDB / PostgreSQL
- BCrypt

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/jedreekPrograms/Clinic.git
   cd Clinic

2. Build the project:
   mvn clean install

3. Run the application:
   mvn javafx:run
